# TPL FVG Editor JSON

Editor JSON desktop per gestire linee, fermate, prezzi e codici del tabellario TPL FVG.

All'avvio l'app chiede se vuoi caricare un file JSON esistente oppure crearne uno nuovo. Da lì puoi modificare le linee, aggiungere o rimuovere fermate, aggiornare le matrici prezzi e codici e scaricare il file JSON finale.
